# 🚀 InstaChat Project Plan (Chat + Call + Social App)

## 🎯 Goal

Build a full-featured communication app like WhatsApp + Instagram with:

* User login/signup
* Send/accept connection requests
* Chat (text + media)
* Voice messages
* Voice/video calling
* Status/Stories

---

## 🧱 Tech Stack

* Frontend: Next.js + Tailwind CSS
* Backend: Node.js + Express
* Database & Auth: Firebase (Firestore + Auth + Storage)
* Real-time: Socket.IO
* Calling: WebRTC

---

## 🧠 System Flow

### 1. Authentication

User → Login/Signup → Firebase Auth → Dashboard

### 2. Connection System

User A → Send Request → User B → Accept → Connected

### 3. Chat System

User → Send Message → Socket → Server → Firebase → Receiver

### 4. Call System

User A → Call Request → User B → Accept → WebRTC Connection → Call Start
